To Compile, use the following commands:

javac AbstractSyntaxTreeInt.java
javac MicroPythonInt_F.java


/******************************/


To Run, use the following commands:

java MicroPythonInt_F [test_file]

Where [test_file] is test1.py, test2.py, test3.py, test4.py, test5.py, or test6.py


/******************************/


Note:
The program will display the contents of the inputted file.

test1.py, test4.py, test5.py, and test6.py require input from the user.
The program will appear stalled until input is specified.
